Clonar y Hacer Fork

Es copiar un proyecto de GitHub a tu computadora local para poder trabajar en él. Se hace usando el comando git clone y te descarga todo el código y el historial del proyecto.

Pull Request

Es crear una copia de un repositorio en tu propia cuenta de GitHub. Sirve para trabajar en un proyecto cuando no tienes permisos directos y luego enviar tus cambios mediante un Pull Request.

Trabajar con Ramas (Branches)

Es una solicitud que haces en GitHub para proponer cambios a un proyecto. Permite que otros revisen, comenten y aprueben tus modificaciones antes de integrarlas al código principal.
