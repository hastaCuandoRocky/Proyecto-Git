¡Aprende Git y GitHub!

¿Cómo crear una cuenta en GitHub?

  1. Accede a GitHub.
  2. Haz clic en "Sign up".
  3. Completa el formulario con tu nombre de usuario, correo electrónico y contraseña.
  4. Confirma tu correo electrónico revisando tu bandeja de entrada y siguiendo el enlace proporcionado.

¿Cómo crear tu primer repositorio?

  1. Inicia sesión en GitHub.
  2. Haz clic en el botón "New repository" o en el signo "+" en la esquina superior derecha y selecciona "New repository".
  3. Rellena los datos:
    • Repository name: Elige un nombre para tu repositorio (por ejemplo, mi-primer-repositorio).
    • Description (opcional): Añade una descripción breve del proyecto.
    • Selecciona si será público o privado.
    • Marca la opción "Initialize this repository with a README" para incluir un archivo README inicial.
  4. Haz clic en "Create repository", y tu repositorio estará listo.

Comandos básicos de Git

  • git init: Inicializa un nuevo repositorio en tu proyecto.
  • git add: Agrega archivos al área de preparación (staging area).
  • git commit -m "mensaje": Registra los cambios en el repositorio con un mensaje descriptivo.
  • git status: Muestra el estado actual del repositorio.
  • git push: Sube los cambios al repositorio remoto (como GitHub).
  • git pull: Descarga los cambios más recientes del repositorio remoto.
  • git log: Muestra el historial de cambios del repositorio.
